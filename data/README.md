# Input documents

Drop these four files here (PDFs preferred; `.md`/`.txt` also works for the career file):

| Expected filename | Content |
|---|---|
| `cv.pdf` | Your curriculum vitae |
| `transcript.pdf` | Academic transcript (completed modules + grades) |
| `career.pdf` (or `career.md`) | Target career path description / job ad / role profile |
| `module_handbook.pdf` | University digital module handbook for your program |

These files are **gitignored** — they contain personal data and are never committed.
